# DielectricPermittivityMeasurementTool
EpsMeasurement.py is a tool that permits measuring the dielectric permittivity of liquid and solid materials by using an open-ended coaxial probe and a network vector analyzer; in particular, the low-cost NanoVNA V1 has been used in this example. As a probe, we used an open-ended section of semi-rigid coaxial cable with a SMA coaxial connector
